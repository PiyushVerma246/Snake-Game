# 🐍 Snake Game

A classic **Snake Game** built using **HTML, CSS, and Vanilla JavaScript**. This project was developed to strengthen my understanding of **core JavaScript concepts** by implementing the complete game logic from scratch without using any external libraries or frameworks.

---

## 🚀 Demo

> Clone the repository and open `index.html` in your browser to play the game.

---

## ✨ Features

- 🎮 Classic Snake Gameplay
- ⌨️ Keyboard Controls (Arrow Keys)
- 🍎 Random Food Generation
- 📈 Live Score Counter
- 🏆 High Score Persistence using Local Storage
- ⏱️ Live Game Timer
- 🔄 Restart Game Functionality
- 💥 Wall Collision Detection
- 🧩 Grid-Based Movement System
- ⚡ Smooth Gameplay using `setInterval()`

---
